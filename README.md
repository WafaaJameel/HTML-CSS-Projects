# HTML-CSS-Projects
Welcome to my web development repository! This project showcases my journey in mastering HTML and CSS, featuring a unique, colorful student portfolio.

## Contents
1.  **Student Portfolio**: A custom-built personal website utilizing advanced CSS techniques.
2.  **Bootstrap Project**: Integration of the Bootstrap framework for responsive design.

## Features Included in Portfolio
* **Fixed Navigation Bar**: A user-friendly menu that stays at the top of the page.
* **Video Background**: An autoplaying, muted, and looped video background with text overlay.
* **Image Effects**: Centered images with rounded corners, CSS saturation filters, and hover-fade overlays.
* **Two-Column Layout**: A structured layout for displaying skills and education.
* **Styled Tables**: Data organized in tables with custom borders, padding, and colored headers.
* **Interactive Elements**: Hover effects on paragraphs and navigation links for an engaging UX.
